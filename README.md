## ✨ Overview

A **minimal yet powerful** single-file web project that renders a **realistic interactive 3D globe** using WebGL technologies.

Built with:
- ⚡ **Zero setup**
- 🌐 **Pure HTML, CSS, JS**
- 🎯 **High performance**

## 🎯 Key Features

- 🌍 Realistic Earth texture rendering  
- 🖱️ Interactive mouse controls  
- 📱 Touch support for mobile devices  
- ⚡ Lightweight & fast (no frameworks)  
- 🎨 Smooth 3D animation using WebGL  
- 📦 Single-file simplicity

## 🧠 How It Works

The project uses:

- **Vanta.js (GLOBE effect)** → handles animation & interaction  
- **Three.js** → powers the 3D rendering engine  
- **Custom texture** → adds realism to the globe
